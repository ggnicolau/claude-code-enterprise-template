# Skill: Research Report

Padrão para produzir relatórios de pesquisa — usado pelo `researcher`.

## Quando usar
Ao pesquisar ferramentas, abordagens, papers ou benchmarks.

## Estrutura do relatório
1. **Pergunta** — o que exatamente está sendo investigado?
2. **Contexto** — por que isso importa para o projeto?
3. **Alternativas** — mínimo 2 opções comparadas
4. **Comparação** — tabela com prós, contras e trade-offs
5. **Recomendação** — escolha clara com justificativa
6. **Fontes** — links para papers, docs ou repos

## Boas práticas
- Nunca recomendar sem comparar alternativas
- Preferir exemplos concretos a explicações abstratas
- Indicar limitações da abordagem recomendada
- Concluir sempre com uma ação clara

# Skill: Market Research

Padrão para pesquisa de mercado, competidores e benchmarks — usado pelo `researcher`, `marketing-strategist`.

## Quando usar
Ao mapear o mercado, analisar competidores ou levantar dados para embasar decisões de produto ou negócio.

## Estrutura da pesquisa

1. **Pergunta central** — o que exatamente precisa ser respondido?
2. **Fontes primárias** — entrevistas, surveys, dados proprietários
3. **Fontes secundárias** — relatórios de mercado, artigos, dados públicos, repositórios GitHub
4. **Análise competitiva** — matriz de features, posicionamento, preços, público-alvo
5. **Benchmarks** — métricas de referência do setor (CAC, LTV, NPS, churn médio)
6. **Síntese** — insights acionáveis, não apenas fatos

## Análise de competidor

| Dimensão | O que avaliar |
|---|---|
| Produto | Features, UX, integrações, limitações |
| Mercado | Segmentos-alvo, geografia, tamanho estimado |
| Negócio | Modelo de preço, funding, crescimento público |
| Posicionamento | Proposta de valor, mensagem principal, diferencial |

## Boas práticas
- Citar fonte para cada dado relevante — sem dados sem origem
- Distinguir dado verificável de estimativa/opinião
- Salvar pesquisa em `docs/research/` com data no nome
- Finalizar com implicações para o produto — pesquisa sem ação é custo
